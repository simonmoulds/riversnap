Tutorials
=========

riversnap tutorials are based on Jupyter notebooks that are hosted on GitHub. 

| **Hydrography Data**
| :doc:`This tutorial <hydrography-data>` will guide you through downloading and preparing some input data to use riversnap.

| **Introduction to riversnap**
| If you're new to the riversnap package, :doc:`this tutorial <introduction>` is the place to get started.

| **PostGIS Integration**
| riversnap can work with PostGIS databases to store and manage large hydrography datasets. :doc:`This tutorial <postgis>` demonstrates how to set up a PostGIS database and use riversnap with it.
